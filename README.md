# Library
This project has been done within the specifications of the course in The Odin Project. The final result can be found here: https://claracanyis.github.io/library/

This is a library App, you can add as many books as you like. The book title, author and pages are mandatory. Feel free to add a link to the book cover for aestetics. You can also mark it as not read or read and indicate the rating of your like.

Once the book is created you can check/uncheck the "read" mark, change the rating or delete the book al together.

By default the app shows all the books regardless of the genre. You can select a genre to show only the books that fit in it. There are also the categories "Fiction" and "Non-fiction" which include several genres. The selector "ALL" will display all the added books again.

# Code
This project has been done with HTML5, CSS and vanila JS for the purposes of practicing the concepts learned in the current studied course.
